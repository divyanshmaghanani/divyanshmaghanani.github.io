Personal Author Website

A case study on building and optimizing my online presence

Project Overview

I built a personal author and portfolio website to showcase my books, certifications, and projects. The main focus was to strengthen my online presence, ensure the site was optimized for search engines, and improve my technical understanding of websites.

Goals & Objectives

Build a professional online hub for my work.

Learn and apply SEO fundamentals (indexing, meta tags, site structure).

Ensure fast performance, mobile responsiveness, and correct technical setup.

Gain basic coding knowledge while managing the project end-to-end.

My Role

Planned the website's structure and design.

Used AI-generated code (HTML/CSS) and edited it to customize sections.

Focused heavily on SEO setup: meta descriptions, robots tag, URL structure, Google Search Console.

Improved technical performance (error fixing, indexing, responsiveness).

Process

1

Planning

Created the sitemap (About, Books, Portfolio, Certifications, Contact).

2

Development Support

Generated the initial code using AI tools, then edited and customized the content.

3

Technical Optimization

Fixed duplicate tags, added robots tag, ensured mobile-friendliness.

4

SEO Setup

Wrote optimized meta descriptions, used proper header tags, and set up indexing in Google Search Console.

5

Launch & Testing

Went live on GitHub Pages and verified that all pages were indexed and working correctly.

Challenges & How I Solved Them

Pages Not Appearing in Google Search

Challenge: Some pages were not appearing in Google search.

Solution: Used Google Search Console's URL inspection, submitted sitemap, and fixed crawlability issues.

Code Structure Issues

Challenge: Duplicate <head> and <body> tags in AI-generated code.

Solution: Learned how to merge them correctly without breaking the layout.

Results & Key Takeaways

Successfully launched a responsive, SEO-optimized personal website.

All pages indexed in Google.

Improved knowledge of SEO and basic coding edits.

Website now serves as the central hub for my books, portfolio, and certifications.

Key Learnings

SEO Fundamentals

Understanding how search engines index content and the importance of proper meta tags.

Code Customization

Learning to edit and customize AI-generated code to meet specific requirements.

Responsive Design

Ensuring the website looks and functions well across all device sizes.
